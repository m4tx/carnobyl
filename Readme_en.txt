--------------------------------------------------------------------------------

The game requires the Java Runtime Environment at version 6 (1.6) or later.
The game should run without any problems on Windows, Linux and Mac.

--------------------------------------------------------------------------------

Carnobyl64k is a game with a top perspective, where the player drives a car, and
his main goal is run over all the pedestrians. At the left-top corner there's
shown amount of smashed and total pedestrians. The whole graphics in the game is
generated by the source code. You can provide a seed, which will be used by the
graphics generator, by the parameter --seed, for example:
 java -jar Carnobyl.jar --seed 16711935
Running the game by this way will give us a pink (magenta) car. The seed must be
a number, it cannot be e.g. text.

The game is open source and it's licensed under GNU GPL v3.

--------------------------------------------------------------------------------

Controls:
- W - accelerating
- S - braking
- A - turning left
- D - turning right

--------------------------------------------------------------------------------

If the game doesn't run smoothly, try to run Java Virtual Machine with
parameter:
 -Dsun.java2d.opengl=true
For example:
 java -Dsun.java2d.opengl=true -jar Carnobyl.jar
